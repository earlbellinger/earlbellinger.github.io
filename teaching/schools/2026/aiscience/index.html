Clustering Clusters

Searching for star clusters with Gaia

Lecturer: Earl Bellinger, Yale University
AI+Science Summer School 2026

Lecture slides

In this lab, you will apply unsupervised machine learning algorithms to observations of the 3D positions and 2D sky motions of stars measured by the Gaia spacecraft (2014-2017) to find a well-known star cluster and compare its properties to a theoretical stellar evolution model.

Before running
Install the packages used for clustering and notebook rendering.

You will also need JupyterLab or Jupyter Notebook to run the lab. If you do not have this on your computer, I personally use Anaconda3 to set it up. You could alternatively ask a coding agent such as Codex or Claude Code to set up the prerequisites for this lab, using this page's URL as context.

HDBSCAN
HDBSCAN stands for Hierarchical Density-Based Spatial Clustering of Applications with Noise. An implementation is available in sklearn, but it is very slow. I recommend installing the python package hdbscan. It can sometimes be tricky to install. Try:

conda install -c conda-forge hdbscan

MyST
The lab uses MyST for beautification! Please also

pip install jupyterlab_myst
