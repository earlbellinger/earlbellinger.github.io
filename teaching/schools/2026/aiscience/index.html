Clustering Clusters

Searching for star clusters with Gaia

Lecturer: Earl Bellinger, Yale University
AI+Science Summer School 2026

  • Slides Lecture slides

In this lab, you will use observations of the positions and motions of stars measured by the Gaia spacecraft in Gaia DR3 (based on data collected from 25 July 2014 to 28 May 2017) to find a well-known star cluster and compare its properties to a theoretical stellar evolution model.

Start here
Download the lab materials from the Google Drive folder, then open lab.ipynb in JupyterLab. Keep all downloaded files in the same directory as the notebook.

  • Files Google Drive folder
  • Notebook lab.ipynb | solutions.ipynb for reference after you have tried the exercises

Before running
Install the optional packages used for clustering and notebook rendering.

HDBSCAN
An implementation is available in sklearn, but it is very slow. I recommend installing the python package hdbscan. It can sometimes be tricky to install. Try:

conda install -c conda-forge hdbscan

MyST
The lab uses MyST for beautification! Please also

pip install jupyterlab_myst

Data used in the lab
The notebook expects these files to be available locally in the same folder.

  • gaia.csv Gaia DR3 observations near the Pleiades.
  • mist_isochrone_112Myr.dat MIST model isochrone for the cluster comparison.
  • known_clusters.dat Catalog of known open clusters for the extension exercise.

Troubleshooting
If hdbscan is difficult to install, you can still inspect the data and plots before the clustering step. The solutions notebook shows the expected output.
