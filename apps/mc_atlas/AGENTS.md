# AGENTS.md

Magellanic Cloud Atlas is a browser-native static web app for exploring the Large and Small Magellanic Clouds in an interactive 3-D canvas scene. It combines OGLE variable-star catalogs, XMC red-clump distance cells, reddening maps, Mira and Anomalous Cepheid distance estimates, Perren cluster data, and MIST-derived synthetic cluster stars into a single generated atlas asset at `public/data/magellanic-clouds.json`, with Brotli/gzip siblings served by `scripts/serve.py`.

The app has no package manager or build step: `index.html` loads `src/app.js` and `src/styles.css` directly, and `python scripts\serve.py 5173` is the normal local server. Most scientific processing lives in `scripts/prepare_data.py`, which parses raw catalogs, assigns reddening, builds compact light/color curve templates, synthesizes clusters, attaches red-clump density, and rewrites the browser data asset; this rebuild can be expensive because ignored local data includes large raw/processed catalogs and about 37 GB of MIST isochrones.

## Project Shape

- Static browser-native atlas app.
- Frontend entry points:
  - `index.html`
  - `src/app.js`
  - `src/styles.css`
- Main data pipeline:
  - `scripts/prepare_data.py`
- Local server:
  - `python scripts\serve.py 5173`

## Generated / Expensive Data

- `public/data/magellanic-clouds.json` is generated by `scripts/prepare_data.py`.
- Compressed siblings are `magellanic-clouds.json.br` and `.gz`.
- `data/`, `diagnostics/`, `.tmp/`, and `papers/` are ignored local support material.
- Full data rebuilds can be slow and may depend on local raw data and optional packages.

## Safe Workflow

- Inspect before editing. This repo often has a dirty worktree.
- Preserve user changes unless explicitly asked to revert them.
- For frontend changes, test with `python scripts\serve.py 5173`.
- For data/schema changes, keep `src/app.js` field indexes aligned with `prepare_data.py` output fields.
- If changing generated data, update raw JSON plus Brotli/gzip siblings together.
- Unless the user requests a change, do not make a change without first confirming. 
- The bundled rg and node.exe in this desktop package is blocked by Windows permissions 
- Playwright is present, but its bundled Chromium binary is not installed; point at the locally installed browser for smoke checks
